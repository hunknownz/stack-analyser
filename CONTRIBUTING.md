# Contributing

## Installation

```sh
git clone https://github.com/specfy/stack-analyser.git
```

```sh
npm install
```

```sh
npm run build:watch
```

- Run the main index.js (do nothing except validate the compilation)

```sh
npm run dev
```

- Run the CLI

```sh
npm run cli
```

- Run the Test

```sh
npm run test
```
